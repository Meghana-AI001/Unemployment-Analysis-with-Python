
📊 Unemployment Rate Analysis

📌 Project Overview:

  Unemployment is measured by the unemployment rate, which represents the percentage of people who are unemployed but actively seeking jobs out of the total labor force. This project focuses on analyzing unemployment trends, especially during the COVID-19 pandemic, to understand its impact on the economy.

🎯 Objectives:
- Analyze unemployment trends over time
- Identify the impact of COVID-19 on employment
- Compare unemployment rates across regions
- Visualize patterns using graphs
- 
📂 Dataset:
The dataset contains:
- Region/State
- Date/Time
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)


🛠️ Technologies Used:
- Python 
- Pandas
- NumPy
- Matplotlib / Seaborn

📈 Analysis Performed:

- Data cleaning and preprocessing
- Trend analysis using line graphs
- Correlation analysis
- Comparison across regions

📊 Key Insights:
📈 Unemployment rates spiked during 2020 due to COVID-19
🏭 Sectors like tourism and hospitality were highly affected
📉 Gradual recovery observed post-2021
🌍 Regional differences in unemployment trends
▶️ How to Run

Install required libraries:
- pip install pandas numpy matplotlib seaborn
- Run the Python script or Jupyter Notebook
- Visualize the results


📌 Conclusion:
    The analysis shows how external shocks like pandemics can significantly affect employment. Understanding these trends helps governments and organizations make better economic decisions.
